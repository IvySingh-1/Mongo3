# Connectly

**Connectly** is a RESTful web application built with **Node.js**, **Express.js**, and **MongoDB (Mongoose ODM)**.  
It demonstrates the implementation of **CRUD (Create, Read, Update, Delete)** operations on a `Chat` resource using **RESTful routing**, server-side rendering with **EJS**, and database persistence with MongoDB.

This project serves as a learning reference for setting up Express applications with Mongoose, defining schemas, handling routes, and performing data operations.

---

## 🚀 Features

- ➕ **Add New Chat** – Create and store new chat entries in the database
- ✏️ **Edit Chat** – Update existing chat content
- ❌ **Delete Chat** – Remove chats from the database
- 📄 RESTful routing with Express
- 🗄️ MongoDB integration with Mongoose
- 🎨 Server-side rendering with EJS

---

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose ODM
- **Templating Engine**: EJS
- **Middleware**: body-parser, method-override
